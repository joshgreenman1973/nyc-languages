In the languages New Yorkers speak at home, sized by number of speakers

American Community Survey, 2022 one-year estimates

Data: U.S. Census Bureau, Decennial Census (1990, 2000) and American Community Survey (2010, 2022)
Font size is proportional to the number of speakers in each decade.

Methodology and sources

What this shows

Each sentence reads "I live in New York City" written in a language spoken by New Yorkers at home. The font size of each sentence is proportional to the number of speakers of that language in a given decade.

Sizing

Font size is scaled using a cube-root function of speaker count. A purely linear scale would make any language under ~100,000 speakers invisible next to English (4+ million) or Spanish (1.8+ million). The cube root preserves the rank order and relative magnitude while keeping smaller languages legible.

Data sources

  • 1990: Decennial Census, tabulated by NYC Department of City Planning (Population Division, April 2004 report).
  • 2000: Decennial Census, Summary File 3, Table PCT010. The last decennial census to include detailed language questions.
  • 2010: American Community Survey (ACS) 2006-2010 five-year estimates, Table B16001, supplemented by ACS 2009-2013 detailed language tables for finer breakdowns.
  • 2022: ACS 2022 one-year estimates, Table B16001.

Important caveats

  • The U.S. Census first asked about "language spoken at home" in 1980, but city-level data from that year is not accessible through the Census API. The earliest reliable NYC-level data is from 1990.
  • Census language categories have changed over time. For example, Bengali was grouped under "Other Indic languages" until the ACS broke it out separately. Similarly, African languages were reported as a single group until recently.
  • From 2010 onward, language data comes from the ACS (a survey with margins of error), not the full decennial count.
  • The 2022 "Yiddish" category includes Pennsylvania Dutch and other West Germanic languages due to category restructuring.
  • "Chinese" is reported as a combined category in most census years. Mandarin/Cantonese splits are only available in the 2009-2013 detailed language tables.

What's missing

NYC is home to an estimated 800+ languages. This visualization includes approximately 35 languages that appear in census tabulations. Many languages spoken by smaller communities, indigenous peoples, and recent immigrants are not captured in the census at all.
